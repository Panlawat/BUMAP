Zone C : Media

Building C1 (อาคาร C1)

อาคาร C1 นอกจากมีห้องเรียน มากมายแล้วยังมีห้องปฏิบัติการ พิกัด: อยู่ใกล้หอสมุดสุรัตน์ โดยเดินเข้ามาอีกสักนิดนึงมองทางขวาไว้ จะมีจุดสังเกตเป็นลานวงกลม

Building C2 (อาคาร C2)

อาคาร C2 มีห้องปฏิบัติการอีกมากมายเช่น สตูดิโอ ห้องจัดรายการวิทยุ ห้องบันทึกเสียง ห้องตัดต่อ ห้องควบคุมกลาง ห้องปฏิบัติการถ่ายภาพ พิกัด: อยู่ใกล้หอสมุดสุรัตน์ โดยเดินเข้ามาอีกสักนิดนึงมองทางขวาไว้ จะมีจุดสังเกตเป็นลานวงกลม

Building C3 (อาคาร C3)

อาคาร C3 โรงละคร Back Box Theater สำหรับให้เรียนรู้การลงมือทำงานจริงเเบบโปรๆ พิกัด: อยู่ใกล้หอสมุดสุรัตน์ โดยเดินเข้ามาอีกสักนิดนึงมองทางขวาไว้ จะมีจุดสังเกตเป็นลานวงกลม

Building C4 (อาคาร C4)

อาคาร C4 คณะดิจิทัลมีเดียและศิลปะภาพยนตร์ จัดหนักจัดเต็มกับสตูดิโอ จะมีสตูดิโอที่ไว้สำหรับการถ่ายทำ รวมถึงยังมีโรงภาพยนตร์สุดพรีเมียม นอกจากนี้ยังมีห้องปฏิบัติการสุดล้ำ เรียกได้ว่า ฝึกจริง ทำจริง ฉายจริงกันเลย พิกัด: อยู่เกือบในสุดของมหาวิทยาลัย ถัดจากตึกของคณะนิเทศ เเนะนำให้นั่งรถรางของ ม.กรุงเทพเข้ามา เเล้วมองด้านขวา สังเกตง่ายๆ จากตึกที่มีสีสันสดใสๆ ตั้งอยู่กลางพื้นที่โล่ง

Building C6 (อาคาร C6 หอสมุดสุรัตน์)

หอสมุดสุรัตน์ นอกจากจะเป็นแหล่งรวมความรู้ทั้งในหนังสือและบนโลกออนไลน์แล้ว ยังมีห้องเรียน Smart Class Room ที่มีอุปกรณ์การเรียนที่ทันสมัยสามารถยืมดูได้ให้กับนักศึกษาด้วย แล้วก็ยังมีโรงภาพยนตร์ Student Lounge, Student Center, International Center, Cyber Center และอีกมากมาย พิกัด: เดินทางต่อจากตึกวิศวะฯ อีกนิดนึง เเล้วมองทางขวาก็จะเห็นหอสมุด
